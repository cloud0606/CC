### 功能设计

- API设计

  登录注册:

  - /signup 用户注册
  - /loginbypw 用户名密码登录
  - /loginbyvc 手机验证码登录（验证码为4位数字）
  - /login 用户登出

   商城相关:

  - /query 查询订单（存在SQL注入）

  - /buy 购买
### 漏洞利用链
  1. /query 存在SQL注入，获取vip用户的手机号
  2. /loginbyvc 使用vip手机号登录，爆破验证码成功登录vip用户账号
